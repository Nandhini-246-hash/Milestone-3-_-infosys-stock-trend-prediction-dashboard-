#  Infosys Stock Trend AI – Stock Market Forecasting Model

## 🔷 Milestone 3 Completion

This project is part of the Infosys Stock Trend AI Internship.  
Milestone 3 focuses on completing model training, evaluation, and dashboard creation for stock price prediction.

---

## 🔷 Project Description

The objective of this project is to predict stock closing prices using Machine Learning techniques.  
Two models, Linear Regression and Random Forest, are implemented and compared to identify the best performing model.

---

## 🔷 Work Completed in Milestone 3

- ✅ Model Training using Linear Regression  
- ✅ Model Training using Random Forest  
- ✅ Model Evaluation using MAE, RMSE, and R² Score  
- ✅ Model Comparison  
- ✅ Historical Stock Price Visualization  
- ✅ Actual vs Predicted Graph  
- ✅ Dashboard Creation  
- ✅ Next Day Stock Price Prediction  

---

## 🔷 Dataset

- Source: Kaggle  
- Features: Open, High, Low, Volume  
- Target: Close  

---

## 🔷 Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🔷 Models Used

- Linear Regression  
- Random Forest Regressor  

---

## 🔷 Evaluation Metrics

- MAE (Mean Absolute Error)  
- RMSE (Root Mean Square Error)  
- R² Score  

---

## 🔷 Results

Random Forest performed better than Linear Regression with:
- Lower MAE  
- Lower RMSE  
- Higher R² Score  

---

## 🔷 Dashboard

A dashboard was created to visualize:

- Historical stock trends  
- Actual vs Predicted values  
- Model performance  
- Comparison results  

---

## 🔷 Conclusion

This milestone demonstrates the successful implementation of Machine Learning models for stock trend prediction.  
Random Forest provided better accuracy and performance compared to Linear Regression.
